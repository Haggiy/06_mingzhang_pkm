## 1. 全局页面规则

`事实`

- MVP 一级入口为：`首页 / 流水 / 资产负债 / 结果视图 / 设置`。
- 页面图稿中使用的 `统计` 等同于 PRD 中的 `结果视图`；本文后续沿用图稿命名 `统计`，但职责按 `结果视图` 理解。
- 首页、流水、资产负债、统计共用账月范围选择器。
- 设置不使用账月范围选择器。
- 所有 `查看来源 / 查看账目 / 相关流水` 都复用流水列表页，不单独新建来源流水页面。
- 首页和流水有全局快速操作入口；资产负债、统计、设置不放全局浮动操作条。
- `账单导入` 只指支付宝 / 微信账单导入；`数据导入` 只指新明账自己的数据文件导入。

`推断`

开发时页面模块可以按以下方式组织：

| 模块 | 页面前缀 | SwiftUI 建议命名 |
| --- | --- | --- |
| 全局组件 | `G` | `MonthRangePickerView` |
| 首页 | `H` | `HomeView` |
| 流水 | `J` | `JournalView` |
| 资产负债 | `B` | `BalanceView` |
| 结果视图 / 统计 | `S` | `StatisticsView` |
| 设置 | `C` | `SettingsView` |

## 3. 全局组件

### G-01 账月范围选择器

`事实`

职责：为首页、流水、资产负债、统计提供统一账月范围。

| 字段 / 控件 | 类型 | 规则 |
| --- | --- | --- |
| 当前账月 | 快捷项 | P0 / P1：选择当前工作账月 |
| 账月列表 | 列表 | P0 / P1：选择单个具体账月 |
| 全部 | 快捷项 | P2：选择全部历史范围 |
| YTD | 快捷项 | P2：选择当年至当前账月 |
| 自定义范围 | 快捷项 | P2：选择起止账月 |
| 应用 | 主按钮 | 应用当前选择 |
| 取消 / 关闭 | 次操作 | 不改变当前范围 |

状态：

| 状态 | 表达 |
| --- | --- |
| 默认 | 显示当前已选范围和可选账月 |
| 自定义范围 | P2：显示起止账月选择 |
| 空账本 | 可显示当前账月，账月列表为空 |
| 无效范围 | 起止账月不合法时禁用应用 |

不承接：账月状态、结账、锁账、核对完成状态。

## 4. 首页页面规格

### H-01 首页

`事实`

职责：展示所选账月范围的收支概览、收支结构、最近账目和快捷入口。

| 字段 / 区块 | 读写 | 数据来源 | 说明 |
| --- | --- | --- | --- |
| 账月范围 | 读 | 全局选择器 | 点击进入 `G-01` |
| 收入合计 | 读 | 流水 / 引擎结果 | 当前范围收入类合计 |
| 支出合计 | 读 | 流水 / 引擎结果 | 当前范围消费和费用类支出 |
| 结余 | 读 | 计算结果 | 收入合计 - 支出合计 |
| 收入结构 | 读 | 统计读模型 | 工作收入、理财收入、其他收入 |
| 支出结构 | 读 | 统计读模型 | 生活必要开支等支出分类 |
| 最近账目 | 读 | 真源流水 | 展示 `manual / import / investment_feed`，不展示 `engine` |
| 快捷按钮 | 操作 | 页面状态 | 打开 `H-04` |

入口：App 启动默认页；底部 Tab。

出口：`G-01`、`H-02`、`H-03`、`H-04`、`J-01`、`J-04`。

状态：

| 状态 | 表达 |
| --- | --- |
| 默认 | 有摘要、结构、最近账目 |
| 无流水 | 摘要为 0，最近账目为空，保留记一笔入口 |
| 计算中 | 摘要区显示加载态 |
| 计算失败 | 摘要区显示错误和重试入口 |

不承接：完整流水管理、资产负债总览、预算、预测、AI、设置维护。

`事实`

| 占比 | 读 | 收入类占收入，支出类占支出 |

| 查看全部 | 操作 | 跳 `H-03` 或 `S-02` 同模板详情 |

状态：默认、无明细、加载失败。

### H-03 分类详情

`事实`

职责：解释一个收入或支出分类在当前范围内的结构、趋势和来源。

| 字段 / 区块 | 读写 | 说明 |
| --- | --- | --- |
| 分类名称 | 读 | 一级分类 |
| 当前范围金额 | 读 | 分类合计 |
| 占比 | 读 | 占收入或占支出 |
| 较上一范围 | 读 | P2：对比值 |
| 月均 | 读 | P2：范围月均 |
| 趋势 | 读 | P2：单月看近 6 月；跨月看范围内按月 |
| 明细构成 | 读 | 类型明细列表和占比 |
| 最近账目 | 读 | 当前分类下真源记录 |
| 查看来源 | 操作 | 跳 `J-06` |

状态：默认、无数据、来源为空、加载失败。

### H-04 快捷菜单

`事实`

职责：承接首页轻量快捷操作。

| 操作 | 目标 |
| --- | --- |
| 记一笔 | `J-04` 新增模式 |
| 导入账单 | 导入来源选择，进入 `J-05` |

## 5. 流水页面规格

### J-01 流水列表

`事实`

职责：正式入账记录的查看、搜索、筛选、新增、编辑和导入整理入口。

| 字段 / 区块 | 读写 | 说明 |
| --- | --- | --- |
| 账月范围 | 读 | 顶部主标题，点击 `G-01` |
| 记录列表 | 读 | 单账月直接展示；跨账月按账月分组 |
| 记录时间 | 读 | 列表显示到分钟，详情保留到秒 |
| 收付手段 | 读 | 记录行第一行 |
| 金额 | 读 | 只用正负号表达方向，不用颜色表达分类 |
| 收付类型 / 类型明细 | 读 | 记录行第二行 |
| 备注摘要 | 读 | 记录行第二行 |
| 搜索 | 操作 | 进入 `J-03` |
| 记一笔 | 操作 | 进入 `J-04` 新增模式 |
| 筛选 | 操作 | 进入 `J-02` |
| 更多 / 导入账单 | 操作 | 进入导入整理流程 |

状态：

| 状态 | 表达 |
| --- | --- |
| 默认 | 显示当前范围记录 |
| 空列表 | 显示空状态和记一笔入口 |
| 跨账月 | 按账月分组，分组行可吸顶 |
| 筛选态 | 显示筛选标签，可清除 |
| 来源筛选态 | 见 `J-06` |
| 加载失败 | 显示错误和重试 |

不承接：收支摘要、结果分析、结账、锁账、预算、预测。

### J-02 流水筛选

`事实`

职责：按用户可见基础字段过滤流水。

| 字段 / 控件 | 类型 | 说明 |
| --- | --- | --- |
| 账月范围 | 选择 | 复用全局账月范围 |
| 时间范围 | 选择 | 不改变账月归属 |
| 收付手段 | 多选 | 系统收付手段 |
| 收付类型 | 多选 | L2 分类 |
| 类型明细 | 多选 | L3 分类，受收付类型约束 |
| 金额范围 | 输入 | 最小值 / 最大值 |
| 备注关键词 | 输入 | 文本匹配 |
| 重置 | 操作 | 清空条件 |
| 应用筛选 | 操作 | 返回 `J-01` 筛选态 |

不包含：record_source、record_kind、engine、investment_feed 等机制字段。

### J-03 流水搜索

`事实`

职责：对当前账月范围内的流水做通用查询。

| 字段 / 控件 | 说明 |
| --- | --- |
| 搜索关键词 | 覆盖金额、收付手段、收付类型、类型明细、备注 |
| 筛选标签 | 显示当前账月范围和可清除条件 |
| 搜索结果列表 | 复用流水记录行 |
| 搜索入口 | 返回 `J-01` 或进入记录详情 |

状态：未输入、搜索中、有结果、无结果、搜索失败。

### J-04 记录详情 / 编辑

`事实`

职责：新增、查看、编辑可直接维护的真源流水记录。

| 字段 | 必填 | 编辑规则 |
| --- | --- | --- |
| 账月 | 是 | 选择 `YYYY-MM` |
| 时间 | 是 | 保留到秒 |
| 收付手段 | 是 | 选择已启用收付手段，可出现 `待补真实账户` |
| 金额 | 是 | 当前科目的带符号发生额 |
| 收付类型 | 是 | L2 分类 |
| 类型明细 | 是 | L3 分类，归属于收付类型 |
| 备注 | 否 | 文本 |
| 来源 | 否 | 只读，展示 `手工记录 / 导入记录` |
| 性质 | 否 | 只读，展示普通记录等 |
| 编辑边界 | 否 | 只读，说明可直接编辑和删除 |

操作：

| 操作 | 规则 |
| --- | --- |
| 保存记录 | 校验必填字段后保存，触发相关账月重算 |
| 删除记录 | 仅 `recordSource = manual / import` 可在流水详情直接删除，需二次确认；`carry_forward` 是记录性质，不单独授权删除 |
| 返回 | 未保存修改时需确认 |

状态：新增、编辑、保存中、保存失败、删除确认、校验失败。

`事实`

| 字段 / 区块 | 读写 | 说明 |

| 待确认数量 | 读 | 未确认候选数 |
| 已忽略数量 | 读 | 被忽略候选数 |
| 候选记录 | 读写 | 可选择、查看、逐条修改 |
| 候选账月 | 写 | 默认按实际时间预填 |

| 收付类型 / 类型明细 | 写 | 可补充或修正 |
| 备注 | 写 | 可补充或修正 |
| 全选 | 操作 | 选择候选记录 |
| 批量修改 | 操作 | 打开 `J-07` |

### B-06 基金交易编辑

`推断`

| 操作 | 规则 |
