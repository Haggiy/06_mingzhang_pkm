# 明账记账科目体系语义与当前实践

> 文档分类：Research / 领域知识来源  
> 说明：本文保留当前 Excel 科目语义分析，用于支撑 PRD 与 Spec，不作为最新产品决议入口。

> 日期：2026-04-16
> 状态：讨论整理版
> 目的：把当前记账科目体系的语义、边界和实际用法整理清楚，作为后续继续梳理 Excel 实践和重建系统的基础。

---

## 1. 文档范围

本文档只整理当前真实在用的记账科目体系，不讨论 App 页面设计，不提前抽象数据库结构，也不追求标准会计口径上的完全对齐。

本文档回答四个问题：

1. 这套科目体系的总体组织逻辑是什么
2. 一级、二级、三级科目分别承载什么语义
3. 当前实践中哪些边界已经明确，哪些仍然带有口袋科目性质
4. 这套体系为什么能支撑当前的复式记账与分析

---

## 2. 总体框架

### 2.1 基础恒等式

当前科目体系建立在以下记账恒等式之上：

`收入 + 负债 = 支出 + 资产`

其中：

- 左边表示资金来源
- 右边表示资金去向

这不是标准借贷字段的直接录入方式，而是当前个人记账实践中的语义框架。

### 2.2 科目层级

当前实践采用三级科目体系：

#### 第一级

- `收入`
- `支出`
- `资产`
- `负债`

#### 第二级

也就是当前主表中的 `收付类型`：

- 支出类
  - `生活必要开支`
  - `文娱游购开支`
  - `财务费用开支`
  - `投资自身开支`
  - `其他必要开支`
- 收入类
  - `工作收入`
  - `理财收入`
  - `其他收入`
- 资产类
  - `资产类支出`
- 负债类
  - `负债类增记`
  - `负债类减记`

#### 第三级

也就是当前主表中的 `类型明细`。

规则：

- `类型明细` 必须唯一归属于一个 `收付类型`
- `类型明细` 不是标签，而是严格的二级细分科目
- 同一个 `类型明细` 不跨 `收付类型` 复用

## 3. 这套科目体系的组织主轴

### 3.1 收入类按来源分

- `工作收入`
- `理财收入`
- `其他收入`

### 3.2 支出类按价值判断分

当前支出类不是单纯按消费对象分，而是带有明确的价值判断主轴：

- `生活必要开支`：衣食住行、日常维持
- `文娱游购开支`：享乐、体验、消费
- `投资自身开支`：成长、提升、长期改善自身状态
- `其他必要开支`：责任、保障、关系义务
- `财务费用开支`：与资金周转和金融行为有关的成本与损失

可以理解为一条从“必须开支”到“享乐开支”再到“成长开支”的组织思路。

### 3.3 资产一级下当前只有一个收付类型

在当前实践中：

- 一级是 `资产`
- 第二级也就是 `收付类型`，当前只有一个：`资产类支出`
- 真正区分资产沉淀形态的是第三级，也就是 `类型明细`

当前 `资产类支出` 下的 `类型明细` 主要包括：

- `现金储蓄`
- `专项储蓄`
- `金融资产投资`
- `个人账户`
- `应收账款`
- `长期待摊费用`
- `其他资产类支出`

### 3.4 负债类按生命周期分

- `负债类增记`：负债的形成与来源
- `负债类减记`：负债的偿还与减少

## 4. 第二级科目语义

### 4.1 支出类

### 4.1.1 生活必要开支

核心语义：

`衣食住行、日常维持型开支`

当前明细包括：

- `伙食费`
- `其他必要家用`
- `交通通信费`
- `房租费`
- `必要着装费`
- `宠物养育费`

说明：

- 这类开支用于维持日常生活的基本运行
- 更强调生存型、维持型、日常性，而不是成长或享乐

### 4.1.2 文娱游购开支

核心语义：

`享乐、体验、消费型开支`

当前明细包括：

- `大件购置费`
- `日常购物费`
- `饮食游乐费`

说明：

- 这类开支更多体现消费欲望、体验需求和生活享受
- `大件购置费` 放在这里，表示其最终会被视为消费性支出，而不是长期资产本身

### 4.1.3 投资自身开支

核心语义：

`为了提升自己、增强能力、改善长期状态而发生的成长型开支`

当前明细包括：

- `教育费`
- `图书费`
- `办公文印费`
- `健身训练费`

说明：

- 这类开支不是享乐，也不是日常生存
- 它们被视为对自身的长期投入
- 当前体系中，`投资自身开支` 的“投资”不是金融投资，而是对自己能力和状态的投入

### 4.1.4 其他必要开支

核心语义：

`不属于衣食住行，不属于享乐，也不属于成长投资，但具有责任、保障、关系维持正当性的必要开支`

当前明细包括：

- `人情费`
- `医疗费`
- `父母亲人赡养费`
- `保险费`

说明：

- 这是当前体系中一个仍带有一定口袋性质的科目
- 但它不是纯垃圾桶，而是承担“责任 / 保障 / 关系义务型开支”的语义
- 当前阶段不建议改名，先稳定其使用边界

### 4.1.5 财务费用开支

核心语义：

`因资金周转、金融服务、金融交易或金融资产处置而发生的成本与损失`

当前明细包括：

- `金融利息支出`
- `金融手续费与罚款`
- `投资亏损`
- `账单套现`

具体说明：

- `金融利息支出`
  - 用于记录贷款、信用卡、借款等带来的利息成本
- `金融手续费与罚款`
  - 用于记录金融操作手续费，以及当前实践中被放入这里的金融相关惩罚性成本
- `投资亏损`
  - 只记录金融资产赎回、卖出、处置时的已实现亏损
  - 未实现浮亏不进这里
- `账单套现`
  - 用于核算信用卡套现过程中的收支和成本
  - 语义上与利息支出接近，但当前实践中保留独立明细

硬规则：

- 已实现收益进 `理财收入`
- 已实现亏损进 `财务费用开支 -> 投资亏损`
- 未实现浮盈浮亏都不进 `流水日记账`

### 4.2 收入类

### 4.2.1 工作收入

核心语义：

`由工作、雇佣关系或劳动活动直接带来的收入与权益增加`

当前明细包括：

- `到手工资`
- `奖金`
- `兼职收入`
- `社保`
- `公积金`
- `非货币性福利`

说明：

- `社保` 与 `公积金` 在语义上首先属于工作收入
- 它们代表工作带来的权益增加
- 随后再通过资产侧进入 `个人账户`

### 4.2.2 理财收入

核心语义：

`金融资产处置时实现的已实现收益`

当前明细包括：

- `权益理财收入`
- `债券理财收入`
- `货币理财收入`
- `商品与衍生品理财收入`

硬规则：

- 只记录赎回、卖出、处置时的已实现收益
- 未实现浮盈不进 `理财收入`
- `PV` 只用于资产分析，不进入主账本

### 4.2.3 其他收入

核心语义：

`不属于工作收入、不属于理财收入，但确实形成收入增加的其他来源`

当前明细包括：

- `其他收入`
- `父母资助`
- `社保返还`

说明：

- 这是一个受限口袋科目
- 有明确语义的，保留具体二级明细
- 没有更合适归属的，进入 `其他收入 -> 其他收入`
- 它不是无限兜底，而是工作收入和理财收入之外的受限兜底收入

### 4.3 资产类

说明：

- 当前资产侧只有一个 `收付类型`：`资产类支出`
- 下面各项均为 `资产类支出` 下的 `类型明细`
- 下面主要解释这些 `类型明细` 在当前实践中的含义

#### 4.3.1 资产类支出

核心语义：

`资金没有被当期消费掉，而是转化为仍然拥有、可持续计量的资产或权益`

它与普通消费开支的根本区别是：

- 普通消费开支：形成当期消耗
- 资产类支出：形成资产沉淀

当前 `资产类支出` 下的 `类型明细` 包括：

- `现金储蓄`
- `专项储蓄`
- `金融资产投资`
- `个人账户`
- `应收账款`
- `长期待摊费用`
- `其他资产类支出`

下面分别说明这些 `类型明细` 的语义。

##### 现金储蓄

核心语义：

`当前实际持有、可直接支配的高流动性资金池`

覆盖范围包括：

- 活期存款
- 现金
- 电子钱包余额
- 可直接开支的货币基金余额等

在当前实践中：

- 主表里的 `资产类支出-现金储蓄` 用于核算当期现金流量变动
- 资产负债表里的 `现金储蓄` 用于核算当前持有的可支配现金余额

##### 专项储蓄

核心语义：

`已经真实划出、为未来特定用途预留的资产资金`

说明：

- 不能只计划，不实际储蓄
- 只有真实进行了储蓄动作，才能计入专项储蓄
- 它不是普通现金，而是已被特定用途占住的资金

在当前实践中，专项储蓄后续可能转向：

- 直接形成消费开支
- 转入 `长期待摊费用`
- 转入其他资产
- 若储蓄过程产生投资收益，则在赎回时一次性确认 `理财收入`

##### 金融资产投资

核心语义：

`进入金融资产持仓体系的资金`

说明：

- 其明细事实维护在 `投资交易记录`
- 主账本只承接账面价值相关变化
- 未实现浮盈浮亏不进主账本
- 已实现收益 / 亏损分别进入 `理财收入` 和 `投资亏损`

##### 个人账户

核心语义：

`归个人所有、可持续计量、但不属于日常现金钱包的社保个人账户类资产权益`

当前子科目包括：

- `公积金`
- `医保账户`
- `养老账户`
- `个人养老金`

说明：

- `公积金 / 医保账户 / 养老账户` 与 `工作收入` 中对应子科目一一对应
- 发工资时，一边记 `工作收入` 增加，一边记 `资产类支出-个人账户` 增加
- `个人养老金` 则可以由主动购买形成
- 这类账户还可能发生：
  - 记账利息结转
  - 提取
  - 支付
  - 返还扣抵

##### 应收账款

当前语义：

`核算对外应收权益`

说明：

- 本轮讨论未继续细化其使用边界
- 当前先保留既有语义，不额外扩展

##### 长期待摊费用

核心语义：

`已经支付或已经形成明确账面资产的支出，其对应的消费权益或使用价值将在未来期间逐步兑现，因此需要分期转为开支`

典型场景包括：

1. 小额但权利以后兑现的待摊费用
   - 如按次数、按额度购买的消费卡
2. 一次支付、分期摊销，且会影响资产负债表的大额待摊费用
   - 如预付房租
3. 房车等大额计入资产负债的待摊消费
   - 如购车首付款与贷款本金形成的待摊消费

与 `专项储蓄` 的区别：

- `专项储蓄`：未来用途的钱先留出来
- `长期待摊费用`：已经形成待摊资产，后续要逐步转为消费

##### 其他资产类支出

核心语义：

`款项已经支付、权利已经兑现、金额较小，但为了更真实地分期反映消费而暂时保留并逐步摊销的过渡性资产类科目`

典型场景：

- 分期免息购买电子产品
- 按年支付的宽带费

与 `长期待摊费用` 的区别：

- `长期待摊费用`
  - 权利以后兑现，或金额足以影响资产负债表
  - 计入资产列报
- `其他资产类支出`
  - 权利已经兑现
  - 不计入资产列报
  - 只是为了摊销口径更合理而暂挂

### 4.4 负债类

### 4.4.1 负债类增记

核心语义：

`负债的形成与来源`

当前明细包括：

- `账单补记`
- `账单分期`
- `现金分期`
- `消费分期`
- `银行借款`
- `亲朋借款`

说明：

- `账单补记` 用于把信用消费汇总后确认为负债
- 其余科目用于区分不同来源和类型的新增负债

### 4.4.2 负债类减记

核心语义：

`负债的偿还与减少`

当前明细包括：

- `账单还款`
- `银行借款还款`
- `亲朋借款还款`

说明：

- 该类科目用于核算负债履约和偿还行为
- 在当前实践中，很多还款类记录会先以固定行形式存在，待实际支付后再补上真实收付手段

## 5. 当前体系中的几条硬规则

### 5.1 投资只认已实现

- 已实现收益进 `理财收入`
- 已实现亏损进 `财务费用开支 -> 投资亏损`
- 未实现浮盈浮亏只用于资产分析，不进入 `流水日记账`

### 5.2 社保与公积金先作为收入，再沉淀为资产

- `社保 / 公积金` 在来源侧属于 `工作收入`
- 在去向侧通过 `资产类支出 -> 个人账户` 进入资产

### 5.3 资产类支出不是消费

- 进入 `资产类支出` 的资金，不视为当期消费完成
- 只有在后续转出、摊销、赎回、支付时，才可能转入开支或收入

### 5.4 其他收入是受限口袋

- 工作收入和理财收入无法解释的收入，进入 `其他收入`
- 有明确语义的保留具体明细
- 没有更合适归属的，进入 `其他收入 -> 其他收入`

### 5.5 其他必要开支当前先稳定语义，不改名称

- 虽然它带有一定口袋性质
- 但当前已明确承载“责任 / 保障 / 关系义务型开支”的语义
- 现阶段先稳定使用边界，不急着改名

## 6. 当前体系评价

当前这套科目体系已经具备以下优点：

1. `主轴清楚`
   - 收入按来源分
   - 支出按价值判断分
   - 资产按沉淀形态分
   - 负债按生命周期分

2. `贴合个人实践`
   - 它不是标准会计模板，也不是普通消费分类器
   - 它直接服务于当前真实可运行的复式记账方法

3. `兼容核算与分析`
   - 能支持 `流水日记账` 的账面发生额核算
   - 也能支撑收支、资产、负债、投资分析

4. `边界大体清楚`
   - `资产类支出` 与消费开支的区别清楚
   - `专项储蓄` 与 `长期待摊费用` 的区别清楚
   - `长期待摊费用` 与 `其他资产类支出` 的区别清楚
   - `理财收入 / 投资亏损` 只认已实现
   - `工作收入 / 理财收入 / 其他收入` 的来源边界清楚

当前仍需后续继续完善的地方主要有：

- 第三级 `类型明细` 的命名和新增原则
- `应收账款` 等尚未完全展开的少数资产科目
- 口袋科目的长期治理方式

但就当前阶段而言，这套二级科目体系已经可以视为稳定可用的当前版本。
